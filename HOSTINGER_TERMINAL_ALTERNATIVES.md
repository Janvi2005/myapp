## **Method 3: SSH (If Terminal Feature Not Available)**

### Option A: PuTTY વાપર (Windows)

1. **PuTTY Download કર:**
   ```
   https://www.putty.org/
   ```

2. **Open PuTTY**
3. **Hostname:**
   ```
   your_vps_ip_or_domain.com
   ```

4. **Port:** 22

5. **Open Click**

6. **Login With:**
   - Username: root
   - Password: તમારો Hostinger password

---

### Option B: Command Prompt (Windows)

```cmd
ssh root@your_domain_name.com
```

Password પૂછશે - તમારો Hostinger password enter કર!
